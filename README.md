# 🌠 MeteorEvent

Плагин для Spigot/Paper 1.16.5 — метеоритный ивент с сундуком и лутом.

## Механика

1. В рандомных координатах мира спавнится сундук
2. Все игроки получают координаты сундука в чат
3. Пока идёт таймер — с неба падают метеориты с частицами и взрываются
4. Когда таймер заканчивается — сундук взрывается и его можно открыть и забрать лут
5. Через 30 секунд сундук исчезает

## Команды

| Команда | Описание |
|---------|----------|
| `/meteorevent start` | Запустить ивент |
| `/meteorevent stop` | Остановить ивент |
| `/meteorevent reload` | Перезагрузить конфиг |

Алиас: `/me`

## Права

| Право | Описание | По умолчанию |
|-------|----------|--------------|
| `meteorevent.admin` | Доступ к командам | OP |

## Конфиг

```yaml
event:
  auto-start-interval: 600  # Автозапуск каждые N секунд (0 = выкл)
  spawn-radius: 200          # Радиус спавна сундука
  chest-timer: 60            # Таймер до открытия (сек)
  meteor-count: 30           # Кол-во метеоритов
  meteor-radius: 15          # Радиус падения метеоритов
  meteor-explosion-power: 2.5
  final-explosion-power: 4.0
```

## Установка

1. Скопируй `.jar` в папку `plugins/`
2. Перезапусти сервер
3. Настрой `config.yml` под себя
4. Запусти ивент: `/meteorevent start`

## Сборка

```bash
mvn clean package
```

Готовый `.jar` появится в папке `target/`.
